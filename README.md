# Vrinda Store Data Analysis

## Overview
Analysis of Vrinda Store 2022 sales data using Microsoft Excel.

## Tools Used
- Microsoft Excel (Pivot Tables, Charts)

## Key Insights
- Women contribute 64% of total revenue
- March recorded highest sales
- Amazon,Myntra and Flipkart channels are max contributing(~80%)
- 92% of orders were successfully delivered
- Maharashtra,Karnataka and Uttarpradesh are the top 3 performing states(~35%)
- Adult age group(25-49 yrs)is max contributing(~60%)
 
## Dataset
Vrinda Store 2022 - Sales Data

## Final conclusion to improve VRINDA STORE SALES:
- Target women customers of age group(25-49yrs) live in Maharashtra,Karnataka and Uttarpradesh by showing ads/offers/coupons available on Amazon,Flipkart and Myntra.
